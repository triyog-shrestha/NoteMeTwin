# NoteMeTwin

A Django-based note-taking application that allows users to create, read, update, and delete notes.

## Features

- Create notes
- View notes
- Edit notes
- Delete notes
- Responsive UI

## Tech Stack

- Python
- Django
- HTML
- CSS
- SQLite

## Setup

1. Create and activate a virtual environment.
2. Install dependencies.
3. Run migrations.
4. Start the development server.

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

## Usage

- Open the home page to see all notes.
- Click **New note** to create a note.
- Open a note to edit or delete it.
